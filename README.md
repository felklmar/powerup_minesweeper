# Minesweeper with powerups
Application/game is classical minesweeper with powerup mode, where you collect tokens and
then spend them on powerups that either flag mines or open empty tile.

 - app/game requires libraries NumPy and Pygame
 - to run the game you need to run main.py using python3 command
 - main.py is located in folder semestral, and other modules are located in src folder
 - folder assets contains font used in menu and folder sprites containing tile icons and game logo
 - module pytest is required to run tests, which are run by simply running pytest command in folder semestral
 - tests are divided into separate files by what module they test
